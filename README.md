## 📘 Overview

A fully responsive, real-time **WebSocket-based Chat Application** developed using **React JS** and **CSS3**.  
This project was created as a demonstration of real-time messaging during my internship at **CodTech IT Solutions**.

---

## 🚀 Features

- ✅ Real-time message exchange using WebSocket  
- ✅ Responsive design for mobile, tablet, and desktop  
- ✅ Auto-scroll to the latest message  
- ✅ Input with Enter key support and send button  
- ✅ Clean, modern CSS3 layout (no timestamps)

## 💡 How to Run Locally

1. Download or clone the repository  
2. Open the terminal and navigate to the project directory  
3. Run `npm install` to install dependencies  
4. Run `npm start` to start the development server  
5. Open `http://localhost:3000` in your browser
